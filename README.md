# Miransas DB (Backend)

Internal database control engine for managing personal projects, database metadata, and secrets.

> ⚠️ This is NOT a SaaS product.  
> This is a private internal tool built for personal infrastructure.

---

## ✨ Features

- Simple and fast Rust backend (Axum)
- PostgreSQL integration (SQLx)
- Project management (internal)
- Database metadata tracking
- Secret storage (isolated logic)
- Admin-only API (Bearer token)
- Health check endpoint
- Migration support

---

## 🧱 Tech Stack

- Rust
- Axum
- Tokio
- SQLx (PostgreSQL)
- Serde
- Tracing
- UUID
- Chrono
